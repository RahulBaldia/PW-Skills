HTML Tags

  1. Div - Used to define a division or a section in an HTML document, often used as a container for other HTML elements.

  2. P - Defines a paragraph in HTML, used to structure text content into separate paragraphs.

  3. A - Creates a hyperlink, allowing users to navigate to another webpage or resource when clicked.

  4. Img - Inserts an image into an HTML document, with attributes specifying the image source, dimensions, alt text, and more.

  5. Ul - Defines an unordered list in HTML, typically used to display a list of items without any specific order.

  6. Ol - Defines an ordered list in HTML, where each list item is numbered sequentially.

  7. Li - Represents a list item in HTML, used within ul or ol elements to define individual items within the list.

  8. H1 to H6 - Define headings of different levels, with h1 being the most important and h6 being the least important. They are used to structure and organize content hierarchically.
