Most common text formatting tags in HTML

  1. Italic Tag
      1) The i tag is used in HTML to render text in italic format.
      2) It is a presentational tag and does not convey any specific semantic meaning.
      3) The use of i tag is primarily for styling purposes, especially when you want to italicize text that doesn't have any specific semantic importance.

  2. Underline Tag
      1) The u tag is used in HTML to underline text.
      2) It is a presentational tag and does not convey any specific semantic meaning.
      3) Similar to the i tag for italics, the u tag is often recommended to be used sparingly, as underlining text can sometimes be associated with hyperlinks.

  3. Bold Tag
      1) The b tag is used in HTML to make text bold.
      2) It is a presentational tag and does not convey any specific semantic meaning.
      3) The use of the b tag is primarily for styling purposes, especially when you want to emphasize text by making it bold
